# Azure Basics Projects ☁️

This repository contains beginner-level Azure cloud projects created using the Azure Portal.

## Project 1: Create Resource Group and Virtual Machine

### Create Resource Group (Azure Portal)

Login to https://portal.azure.com

Search Resource groups

Click Create

Fill details:

Subscription: Default

Resource group name:

rg-azure-basics


Region:

UK South (or any nearby region)


Click Review + Create

Click Create

✅ Resource Group created

### Create Virtual Machine

In Azure Portal → Search Virtual Machines

Click Create → Azure Virtual Machine

Basics Tab:

Subscription: Default

Resource group:

rg-azure-basics


Virtual machine name:

vm-basics-01


Region:

UK South


Image:

Windows Server 2019 Datacenter


Size:

Standard B1s (Free / Low cost)


Username:

azureuser


Password : StrongPassword@123

⚠️ Save username/password safely

Click:

Next → Review + Create

Click Create

⏳ Wait 2–5 minutes
✅ Virtual Machine created


### Steps Performed

1. Logged into Azure Portal
2. Created a Resource Group named `rg-azure-basics`
3. Created a Virtual Machine named `vm-basics-01`
4. Verified VM running status
